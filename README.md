# nanocarrier-ai

나노전달체 연구에 AI/ML을 적용하기 위한 실습 저장소.

## Environment

This project uses WSL2 Ubuntu and Miniconda.

## Setup

```bash
conda create -n nanocarrier-ai python=3.11
conda activate nanocarrier-ai
pip install -r requirements.txt
```

## Folder structure

```text
nanocarrier-ai/
|-- README.md
|-- requirements.txt
|-- notebooks/
|-- data/
|-- src/
`-- results/
```

## Goal

- Practice Python for machine learning
- Build basic regression models
- Organize data, notebooks, and results with GitHub
